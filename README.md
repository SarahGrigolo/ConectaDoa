ConectaDoa – Aplicativo de Doações

O ConectaDoa é um aplicativo desenvolvido como trabalho escolar com o objetivo de conectar doadores a instituições que necessitam de roupas, alimentos, produtos de higiene e outros itens essenciais.O app facilita o processo de doação, tornando mais rápido e acessível para quem deseja ajudar.

 - Funcionalidades

Cadastro e login de usuários

Listagem de instituições disponíveis

Cadastro de doações

Visualização de doações realizadas

Interface simples e intuitiva

 - Download do APK

Clique no link abaixo para baixar a versão mais recente do aplicativo:

Download [ConectaDoa.apk](https://github.com/SarahGrigolo/ConectaDoa/releases/tag/tag1)

- Como instalar o APK

Baixe o arquivo .apk na seção Releases do repositório

No seu celular Android ou Emulador, abra o arquivo

Caso solicitado, habilite a opção:

Configurações → Segurança → Instalar apps desconhecidos

Instale normalmente
